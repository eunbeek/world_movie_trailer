
# World Movie Trailer
![image](https://github.com/user-attachments/assets/55d9372c-80ff-4f0d-be4c-a7c613036b13)
**World Movie Trailer** is a mobile application that allows users to browse trailers for released and upcoming movies worldwide. The app fetches movie data and trailer information from various sources and provides a seamless experience for users to discover and watch movie trailers.

- Google Play Store : https://play.google.com/store/apps/details?id=com.sunnyinnolab.worldMovieTrailer
- iOS Apple Store : https://apps.apple.com/us/app/world-movie-trailer/id6670228768
## Features

- Browse movie trailers by country.
- View trailers for upcoming and released movies.
- Weekly updates with new movie data.
- Offline access to previously fetched trailers.
- Multilingual support (English, Korean, Japanese, Chinese, etc.).

## Technologies

- **Flutter**: Frontend development framework.
- **Firebase Storage**: Store and access movie data in JSON format.
- **Hive**: Local storage for offline access and caching.
- **SharedPreferences**: Cache data for quick access between Firebase updates.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/world-movie-trailer.git
    cd world-movie-trailer
    ```

2. Install dependencies:
    ```bash
    flutter pub get
    ```

3. Run the app on your preferred device/emulator:
    ```bash
    flutter run
    ```

## Usage

1. Select your country from the list to view available movie trailers.
2. Explore released and upcoming movies.
3. Bookmark your favorite trailers for easy access.
4. Data is updated weekly, and cached data is available offline.
